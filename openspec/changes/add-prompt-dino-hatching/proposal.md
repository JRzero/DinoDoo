# Add Prompt-Based Dino Hatching

## Why

The selected product direction changes the middle H5 navigation entry from voice to `孵化`. Children or parents should be able to describe the dinosaur they want and hatch a custom little dinosaur from an egg-themed screen.

## What Changes

- Add a main `孵化` H5 screen following the selected dinosaur-island visual direction.
- Let users describe the desired dinosaur by typing, voice input, or prompt chips.
- Save locally hatched dinosaurs into the artwork/gallery experience.
- Keep story voice controls inside the story screen instead of making voice a main nav destination.

## Non-Goals

- No mobile shell.
- No user account system.
- No paid generation pipeline.
- No public sharing workflow.
