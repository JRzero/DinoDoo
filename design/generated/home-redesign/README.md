# Home Redesign Asset Pack

Generated for the DinoDoo H5 home screen redesign.

## Runtime Assets

Use these files from `apps/h5/assets/game-elements/home-redesign/` when wiring the H5 page:

| File | Purpose |
| --- | --- |
| `home-empty-redesign.png` | 390 x 684 empty island background for the H5 content area |
| `dino-xiaobao-redesign.png` | Transparent Xiaobao orange T-rex character |
| `dino-adai-redesign.png` | Transparent Adai boy triceratops character |

## Design Sources

This folder keeps generated source images and processed copies:

- `*-source.png`: original ImageGen outputs before chroma-key removal or resizing.
- `*-redesign.png`: transparent processed character/object assets.
- `home-empty-redesign-full.png`: 390 x 844 full background reference.
- `asset-preview.png`: visual preview sheet for quick review.

## Notes

- Character and object assets were generated on a magenta chroma-key background, then processed to PNG with alpha.
- Badge assets are intentionally blank. Add Chinese labels separately so text remains accurate and editable.
- The preview mock is only an asset relationship check, not the final H5 page layout.
